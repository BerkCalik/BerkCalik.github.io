# Florence Nightingale Hastanesi - Parking Automation

A parking automation system developed for Florence Nightingale Hastanesi to digitize vehicle entry, exit, ticketing, and parking management workflows.

The project was designed around handheld terminal usage in the parking area. Operators could authenticate from the Android application, register vehicle entries and exits, search vehicles by license plate, and print parking receipts directly from the device's built-in printer through native terminal integration.

The system also included a .NET Core backend API and a .NET Core based management panel for operational tracking, reporting, role-based access, and subscriber management.

## Overview

The main goal of the project was to replace manual parking workflows with a faster and more traceable digital system.

The mobile application enabled field staff to handle daily parking operations directly from handheld terminals, while the admin panel gave back-office users access to reports, user management, subscriber records, and role-specific screens.

The system focused on:

- Fast vehicle entry and exit operations
- License plate based vehicle lookup
- Secure user authentication
- Native handheld terminal integration
- Built-in printer receipt output
- Centralized parking records
- Role-based operational management
