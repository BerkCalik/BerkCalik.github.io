# FreeBird Airlines – B2C & B2B Platforms

FreeBird Airlines provides a unified digital ecosystem for modern aviation services, catering to both direct consumers and corporate partners through dedicated B2C and B2B platforms.

The B2C platform focuses on a seamless, high-speed booking experience for travelers, while the B2B platform offers robust management tools for travel agencies and corporate clients.

---

## 🚀 Overview

The project aims to modernize the air travel experience by providing two specialized web applications that share a consistent design language while serving different user needs.

- B2C (Business to Consumer): A traveler-facing application optimized for performance and international reach.
- B2B (Business to Business): A corporate-facing portal designed for efficiency, high-volume data handling, and agency management.

## ✨ Key Features

### Global Reach

Both platforms feature full internationalization support, allowing users to navigate and book in their preferred language across different regions.

### Interactive Experience

Smooth transitions and interactive elements powered by Framer Motion provide a premium feel throughout the booking flow and user dashboard.

### Agency Management (B2B)

Advanced tools for corporate partners to manage bookings, track transactions, and handle bulk operations efficiently.

### Optimized Search (B2C)

A fast and intuitive flight search engine built with server-side rendering for maximum performance and search engine visibility.

---

## 🏗 Architecture & Technical Approach

The project utilizes specialized environments to ensure that each platform uses the best tool for its specific purpose while maintaining architectural consistency.

- Server-Side Rendering: Leveraged in B2C for better initial load speed and optimization.
- Client-Side Performance: Optimized in B2B for a fast and reactive administrative experience.
- Containerization: Standardized development and production environments for reliability.

---

## 🛠 Tech Stack

### B2C Platform (Customer Facing)

- Next JS
- Tailwind CSS
- Shadcn UI
- Zustand (state management)
- next-intl (internationalization)

### B2B Platform (Corporate Facing)

- React JS
- Vite
- Tailwind CSS
- Shadcn UI
- Zustand (state management)
- react-i18next (internationalization)

### Shared Infrastructure & Animation

- CI/CD Pipelines (automated deployment)
- Docker (containerization)
- Framer Motion (animations)

---

## ⚡ Performance & Delivery

- Containerized deployment using Docker for consistency across different environments.
- Automated CI/CD pipelines to ensure rapid and reliable delivery of updates.
- Modern state management with Zustand to reduce re-render overhead and complexity.
